# micro:bit の input/output を教材に活用するイメージ

## このページについて

micro:bit を教材として活用する際に使える input/output をまとめている。

## INPUT について

* センサー
  * 加速度
  * 温度
  * 地磁気
  * コンパス
  * 明度（明るさ）
* ボタン
  * A/B/同時押し
* 端子（ピン）入力
  * P0,1,2 触る（通電）を入力として使える
* 無線（受信）
  * 他のmicro:bitからの無線の受信

## OUTPUT について

* LED
  * 5x5 のLED
* 無線（送信）
  * 他の micro:bit への無線送信
